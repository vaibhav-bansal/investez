# Archived Vercel Configurations

These Vercel configuration files are archived for reference.

**Migration Date**: January 2026

## Contents

- `frontend-vercel.json` - Dashboard frontend Vercel config
- `marketing-vercel.json` - Marketing website Vercel config

## Why Archived?

The project has been migrated entirely to Railway for simplified deployment.
These files are kept for historical reference and in case a migration back is needed.

## Migration

If you need to redeploy to Vercel:

1. Copy the appropriate `*-vercel.json` back to its original directory
2. Rename to `vercel.json`
3. Follow Vercel deployment instructions
4. Update API URLs in environment variables

## Current Deployment

All services now run on Railway. See [RAILWAY_DEPLOYMENT.md](../../RAILWAY_DEPLOYMENT.md) for details.
